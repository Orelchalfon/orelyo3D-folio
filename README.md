# Orel Chalfon - 3D Portfolio

A modern 3D portfolio website built with React, Three.js, and Vite. Features interactive 3D models, smooth animations, and a responsive design.

## About

This portfolio showcases my work as a fullstack developer specializing in:
- React.js/TypeScript with Vite and Next.js
- MongoDB, Node.js, Express
- Supabase and Prisma
- Modern UI/UX design principles

## Tech Stack

- **Frontend**: React 18, React Router
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Styling**: Tailwind CSS
- **Animations**: React Spring, Framer Motion
- **Build Tool**: Vite
- **Contact Form**: EmailJS

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/orelchalfon/3D_portfolio.git
cd 3D_portfolio
```

2. Install dependencies:
```bash
pnpm install
```

3. Create a `.env` file in the root directory and add your EmailJS credentials:
```env
VITE_APP_EMAILJS_SERVICE_ID=your_service_id
VITE_APP_EMAILJS_TEMPLATE_ID=your_template_id
VITE_APP_EMAILJS_PUBLIC_KEY=your_public_key
```

4. Run the development server:
```bash
pnpm dev
```

5. Build for production:
```bash
pnpm build
```
